2Given an array of integers, every element appears 

3three times except for one, which appears exactly once.

11Solution: 

1232 bits for each integer. 

13Consider 1 bit in it, the sum of each integer's corresponding bit 

14(except for the single number) 

15should be 0 if mod by 3. Hence, we sum the bits of all 

16integers and mod by 3, 

17the remaining should be the exact bit of the single number.

22def single_number2(nums): 

23 ones, twos = 0, 0 

24 for i in range(len(nums)): 

25 ones = (ones ^ nums[i]) & ~twos 

26 twos = (twos ^ nums[i]) & ~ones 

27 return ones
